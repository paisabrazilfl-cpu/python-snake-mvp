Python Snake Game MVP
=========================

This is a full MVP implementation of the classic Snake game using the Pygame library.

Installation
------------

To install the required dependencies, run the following command in your terminal:

    pip install -r requirements.txt

Running the Game
----------------

To run the game, execute the following command in your terminal:

    python main.py


Graphics
---------

The game features bright, cartoonish graphics inspired by the style of Mario and the bold color palette of Street Fighter.
