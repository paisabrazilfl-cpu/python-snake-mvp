# Project Overview

This is a Python Snake game MVP, inspired by the classic games Street Fighter and Mario. The game is built using the Pygame library and features a 2D ultra-realistic design.

# System Requirements

To run the game, you will need to have Python 3.10 or later installed on your system. You will also need to have the Pygame library installed.

# Installation

To install the game, simply run the following command in your terminal:

pip install -r requirements.txt

# Running the Game

To run the game, simply run the following command in your terminal:

python main.py

# Controls

The game can be controlled using the arrow keys to move the snake, and the ESC key to quit.

# Code Structure

The game is structured into several sections in the main.py file. The game loop, collision detection, and exit handling are all implemented in this file.

# Assets

The game uses two assets: a snake head image and a food image. These images are stored in the assets folder and are referenced in the main.py file.

# Conclusion

This is a basic implementation of a Python Snake game MVP. The game can be further developed and improved upon by adding more features and assets.
